#불닭낋여오너라


팀원 : 전예린, 박수민, 박준석, 길단유
역할 : 전예린-디자인, 박수민-자료조사, 박준석-테스트, 길단유-기획
기능 : 불닭 레시피를 크리미한, 시원한, 단짠단짠, 화끈한, 든든한 총 5가지 카테고리로 나누어 추천해주는 사이트이다. 불닭을 즐겨 먹는 10대·20대 사용자들이 간단한 재료만으로도 다양한 방식으로 불닭을 즐길 수 있도록 레시피 아이디어를 제공한다.
기술 : HTML을 사용해 페이지 구성, 텍스트 및 이미지 배치를 설계하였으며, CSS는 HTML 파일 내부에서 <style> 태그로 작성하여 색상, 정렬, 레이아웃, 버튼 등의 디자인 요소를 적용하였다

CODE 구조 : HTML 파일을 기반으로 각 페이지가 구성되어 있으며, index.html에서 카테고리별 레시피 페이지로 이동하는 구조를 사용하였다.
느낀점 : 길단유-이 사이트가 더 개선되어 삼양에 닿았으면 좋겠다. 박준석-이번 프로젝트를 통해 웹 구조와 반응형 디자인, Git 브랜치 관리 등 기본 개념을 실제로 적용하며 시행착오 속에서 성장과 성취감을 느낄 수 있었다. 전예린-이번 활동을 통해서 GITHUB가 여러 방면에서 사용된다는 것을 알게되어, 다양한 기업에서 진행하는 해커톤을 참여해볼 것이다. 박수민-불닭 레시피를 제공하는 웹페이지를 만드는 과정에서 vscode 및 깃허브 사용법에 대한 전반적인 이해를 높였으며, 완성된 웹페이지를 보며 만족감을 느꼈다. 부족한 부분을 조금 더 개선해서 실제로도 유용하게 활용되었으면 좋겠다.
